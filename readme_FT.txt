#### ORGANIZATION *.sv FILES FOR RTL ####
In the directory ./rtl, there are the files .sv for the description of the core.
In the subdirectory ./rtl/rtl_FT, there the files for the FT version of the core.
In the subdirectory ./rtl/original_files_to_not_include, there the original files replaced by the FT version of those components.

#### IMPORTANT ####
Keep in mind to adjust the manifest if different files are used.
